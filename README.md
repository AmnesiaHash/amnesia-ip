# AmnesiaIP

Быстрая и безопасная проверка вашего IP-адреса. Одностраничный сервис с минималистичным фиолетовым дизайном.

## Структура

```
amnesia-ip/
├── index.html   — разметка страницы
├── style.css    — стили и анимации
└── script.js    — логика получения данных
```

## Запуск локально

Откройте `index.html` напрямую в браузере или используйте локальный сервер:

```bash
# Python
python -m http.server 8080

# Node.js (npx)
npx serve .
```

Затем откройте `http://localhost:8080` в браузере.

## API

- **Основной:** [ipwho.is](https://ipwho.is/) — геоданные, ISP, ASN, часовой пояс
- **Fallback:** [ipapi.co](https://ipapi.co/) — при недоступности основного API
- **IPv6:** [api64.ipify.org](https://api64.ipify.org/) — определение IPv6-адреса

## Функции

- Автоматическое определение IPv4, IPv6, страны, города, региона, провайдера, ASN, организации, координат, часового пояса
- User-Agent и тип подключения (клиентские данные)
- Копирование IP в буфер обмена
- Обновление данных без перезагрузки страницы
